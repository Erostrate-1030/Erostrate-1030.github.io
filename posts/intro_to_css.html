CSS 基础

  • Cascading Stylesheets 层叠样式表
    • 告诉浏览器如何指定样式、布局
  • selector: 对应 html 中的 <p>

添加CSS

  • 外部样式表（常用）
    • CSS 保存在 .css 文件中
    • HTML 中使用 <head><link> 进行连接
  • 内部样式表（可以使用）
    • 不使用外部CSS
    • HTML 中使用 <head><style> 标签放置 CSS
  • 内联样式（不推荐使用，维护成本高）
    • 仅影响一个元素
    • HTML 元素的 style 属性添加

CSS 代码

CSS 中选择 HTML

直接指向标签

指向类（不论任何标签，指向所有的这个类）

指向 id （一般 id 指 HTML 中独一无二的一个）

HTML <div class="box">

输入 .box1 自动生成 <div class="box1"></div>

HTML <div class="container">

用 container 类包裹整体内容，方便排版。

CSS 特征设置

关于页面占比/对齐相关的设置有：

  • width: 设置总体占宽度
  • margin: auto 居中
  • padding: 缩进

关于字体的设置有，详见：

  • font-weight: 字体粗细
  • font-family: 字体选择
    • font-family: Arial, Helvetica, sans-serif
      • Arial 字体会优先选择，如果找不到则使用 Helvetica
      • 无衬线字体
    • font-family: "Gill Sans", sans-serif;
      • 引号是因为字体名称中间有空格
    • font-family: "Times New Roman", serif;
      • 有衬线字体
    • font-family: "Andale Mono", monospace;
      • 等宽字体

相关网站：

  • 查询字体占比：https://www.cssfontstack.com
  • 免费字体选用：https://fonts.google.com
  • 字体设计：https://thetype.com

CSS 盒子

  • Margin: 外边距
    • 当两个 Margin 相邻，塌陷为一个 Margin，且为两者中最大的 Margin
    • 上右下左（顺时针方向）
  • Padding: 内边距
    • 同 Margin
